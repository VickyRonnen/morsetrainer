Overview of the lessons

Morse code uses dots and dashes. When you want to speak out morse code, use dit for the dot and dah for the dash. The dash is three times longer than the dot, so you can hear the difference.

This page contains two expandable sections. Each section includes a table listing lesson numbers, the new characters, and their Morse code.
